# BMI Calculator

A responsive BMI (Body Mass Index) calculator web application built with vanilla JavaScript, HTML, and CSS. This project is part of the Frontend Mentor challenges and helps users calculate their BMI using both metric and imperial units.

## Live Demo

You can try out the BMI calculator here: [https://umtcnzn.github.io/bmi-calculator](https://umtcnzn.github.io/bmi-calculator)

## Features

- Calculate BMI using metric (kg/cm) or imperial (st-lbs/ft-in) units
- Dynamic unit switching
- Real-time BMI calculation
- Displays ideal weight range based on height
- Responsive design that works on desktop and mobile devices
- Clear and intuitive user interface
- No framework dependencies - built with pure JavaScript

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- No frameworks or libraries

## Local Development

To run this project locally:

1. Clone the repository
   ```bash
   git clone https://github.com/umtcnzn/bmi-calculator.git
   ```
2. Navigate to the project directory
   ```bash
   cd bmi-calculator
   ```
3. Open `index.html` in your browser
